# Koala Habit Product Brief

## Purpose

Koala Habit helps elementary school children combine learning, movement, and entertainment into a healthy family habit loop. Children earn entertainment time and companion growth by completing meaningful tasks. Parents guide the goals and confirm outcomes without turning every day into a negotiation.

## Primary Users

- Parent: 30-50 years old, wants children to build habits without daily conflict, and wants visible growth feedback.
- Child: 6-12 years old, with the first version focused on elementary school summer habit building.

## First Experience

The iPad app is the first device target. The first screen should feel calm, playful, and easy to scan from a table or couch distance. A child should understand today's mission board without reading long instructions.

## MVP Loop

1. Parent creates or accepts today's task plan.
2. Child completes learning, movement, and life missions.
3. Child submits completion.
4. Parent confirms the result.
5. App unlocks entertainment time and companion progress.

## Initial Domains

- Families and children
- Daily missions
- Parent confirmations
- Reward time
- Companion growth
- Streaks and weekly summaries

## MVP Scope

### Login

- Parent account uses Apple Sign In for the first version.
- Google Sign In is planned for a later phase.
- Guest mode is optional.
- Parents create child profiles with name, age, koala avatar, and grade.
- Children log in with a numeric PIN created by the parent.

### Child Today Page

- Greeting, for example `Good Morning, Caitlyn!`
- Today Energy, for example `120`.
- Daily tasks for summer learning and movement.

### Default Tasks

- English Reading: read 20 minutes.
- Chinese: poem or Chinese reading.
- Math: 10 questions.
- Piano: practice 20 minutes.
- Soccer: soccer training.

### Task Detail

- Goal details.
- Complete button.
- Optional photo upload placeholder.
- Optional audio upload placeholder.
- Lightweight AI encouragement using preset copy and TTS later.

### Growth Tree

- Tasks grow a tree through five stages: seed, sprout, young tree, blooming tree, and forest.
- Completion feedback should include glow, leaves, and stars.

### Parent Console

- Task management with default templates.
- Add task fields: name, category, goal, reward energy, and repeat cycle.

### History

- Today completion.
- Weekly completion rate.
- Streaks such as `Reading Streak: 7 days`.

### Reminders

- Local notification reminders, for example `Reading time is here`.

### AI Encouragement

- First version uses preset encouragement copy.
- Text-to-speech playback can be added after the core flow is stable.
